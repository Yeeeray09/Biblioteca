# Biblioteca

## Descripción

Este trabajo ha sido creado con el fin de crear un menú interactivo en el cual se puedan hacer reserva, listar usuarios y devolverlos. Lo hemos separado 4 distintos apartados, el controlador, la vista y el README (Hecho por Gabriel) y el modelo (Hecho por Yeray). Al ejecutar el trabajo saldrá un menú interactivo en el cual tiene las opciones de Listar Libros, Resumen de Usuarios, Realizar Préstamo, Devolver Libro y Salir. Dentro de cada apartado hay más opciones como por ejemplo en Realizar Préstamo se pueden realizar un alquiler de un libro pero previamente es necesario saber su ISBN que se encuentra seleccionando la opción 1.
